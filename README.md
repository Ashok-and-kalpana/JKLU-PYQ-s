# 📘 JKLU Previous Year Question Papers (PYQs) Repository

Welcome to the **JK Lakshmipat University (JKLU), Jaipur PYQs Repository** 🎓.  
This project is an initiative by students, for students — to build a **central hub of previous year question papers, quizzes, assignments, and practice sheets** for reference and preparation.  

The main idea is simple:  
👉 Instead of every student searching individually for old papers, we maintain a **single open repository** on GitHub that can be accessed anytime, anywhere.  
👉 This helps juniors prepare better and seniors contribute their collected resources for the community.  

---

## 📂 Repository Structure

The repository is neatly organized **subject-wise** and **department-wise**.  
Each course has its own dedicated folder, and inside those folders you will find question papers (in PDF/Image formats).





### 📌 Inside Each Subject Folder
- **Previous Year Question Papers (PYQs)**  
- **MidSem & EndSem Papers**  
- **Quizzes and Assignments**  
- **Any other useful practice sheets**  

---

## 🗂️ File Naming Conventions

To keep the repo clean and easily searchable, we follow these naming rules:

- `PYQ-<Year>-<ExamType>.pdf`  
  Example: `PYQ-2023-EndSem.pdf`  

- `Quiz-<No>-<Year>.pdf`  
  Example: `Quiz-2-2024.pdf`  

- `Assignment-<Topic>-<Year>.pdf`  
  Example: `Assignment-LinkedList-2022.pdf`  

This makes it easier for everyone to quickly find the right file.  

---

## 🙌 How to Use This Repository

1. **Browse folders** → Go to the subject/course you want.  
2. **Open PDFs** → Directly view question papers on GitHub or download them.  
3. **Practice** → Use them as references to understand exam patterns and prepare better.  

---

## 📤 Contribution Guide

Currently, this repository is **centrally maintained**.  
Students can contribute by sharing papers with the maintainer, who will upload them properly.  

👉 How you can share:  
- Email the PDFs/Images  
- Share via Google Drive/WhatsApp  
- Mention subject + year + exam type clearly  

⚠️ *Direct PRs are not open right now to avoid disorganized uploads.*  

---

## 🚀 Vision of This Project

- 📚 To create a **free, open, and structured academic resource** for JKLU students.  
- 🤝 To encourage a **culture of sharing and collaboration** among students.  
- ⏳ To **save time** for juniors so they can focus on preparation instead of searching.  
- 🎯 To make this repo a **one-stop solution** for all academic PYQs and quizzes at JKLU.
