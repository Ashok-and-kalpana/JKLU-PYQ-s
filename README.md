# 📘 JKLU Previous Year Papers (PYQs)

This repository contains **Previous Year Question Papers (PYQs), quizzes, and practice sheets** for JK Lakshmipat University students.  
All files are organized subject-wise for easy access.

---

## 📂 Structure
- `CSE/Data-Structures/` → PYQs for Data Structures
- `CSE/Operating-Systems/` → PYQs for Operating Systems
- `ECE/Signals-and-Systems/` → PYQs for Signals & Systems
